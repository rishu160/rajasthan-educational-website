# 🏛️ Rajasthan Digital Education Platform

## Government of Rajasthan Remote Classroom for Rural Colleges

A comprehensive digital education platform with **AI Chatbot**, **Payment Gateway**, and **Advanced Features** developed for the Government of Rajasthan to provide remote classroom facilities for rural colleges.

## 🚀 Live Demo
- **Firebase**: [https://rajasthan-education-platform.web.app](https://rajasthan-education-platform.web.app)
- **GitHub Pages**: [https://rishu160.github.io/rajasthan-educational-website/](https://rishu160.github.io/rajasthan-educational-website/)
- **Admin Dashboard**: [Live Admin Panel](https://rajasthan-education-platform.web.app/admin/dashboard.html)

## ✨ New Features Added

### 🤖 AI Chatbot System
- **Bilingual Support**: Hindi/English conversations
- **Voice Assistant**: Speech-to-text and text-to-speech
- **Smart Responses**: OpenAI GPT integration
- **Quick Actions**: Fast navigation buttons
- **Analytics**: Usage tracking and insights
- **24/7 Support**: Always available for students

### 🔍 Smart Search Engine
- **Real-time Search**: Instant results while typing
- **Multi-category**: Courses, colleges, services
- **Smart Matching**: Title, description, location search
- **Quick Navigation**: Click to go directly to pages

### 💳 Payment Gateway Integration
- **Razorpay Integration**: Secure online payments
- **Multiple Options**: Cards, UPI, Net Banking, Wallets
- **EMI Support**: Flexible payment plans
- **Government Discounts**: Special schemes for rural students
- **Payment Tracking**: Complete transaction history

### 📊 Admin Dashboard
- **User Management**: Student and teacher accounts
- **College Management**: Add/edit college information
- **Payment Analytics**: Revenue and transaction tracking
- **Chatbot Analytics**: Conversation insights
- **Real-time Stats**: Live platform statistics

### 🔥 Firebase Backend
- **Authentication**: Secure user login/registration
- **Firestore Database**: Real-time data storage
- **Cloud Functions**: Serverless backend APIs
- **File Storage**: Document and video uploads
- **Analytics**: User behavior tracking

## 🚀 Quick Start

1. Clone the repository
```bash
git clone https://github.com/rishu160/rajasthan-educational-website.git
```

2. Open `index.html` in your browser

3. Navigate through different sections:
   - Home - Overview and statistics
   - Colleges - Partner college information
   - Courses - Available course catalog
   - Services - Digital services offered
   - Contact - Get in touch

## 📱 Pages

- `index.html` - Main homepage
- `login.html` - User authentication
- `college-details.html` - Detailed college information
